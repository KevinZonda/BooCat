# Bookie

A universal book search platform which allows you to search several providers by just one-click.

## Supported Providers

- [x] Z-Library
- [x] Library Genesis
- [x] Open Library
- [x] Memory of the World
- [x] The Online Books Page by UPenn

## Tech Stacks

- [x] .NET 6
- [x] Azure Function App
- [ ] ASP.NET Core
- [ ] Blazor
- [ ] Razor
- [ ] WebAssembly
